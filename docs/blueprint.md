# **App Name**: Glitch Coffee

## Core Features:

- Minimalist Sticky Navbar: A sticky navigation bar with the 'GLITCH COFFEE' logo on the left and navigation links on the right, separated by thin line borders.
- Editorial-Style Hero Section: Hero section with the headline 'STEP INTO THE STATIC', a sub-headline describing the coffee shop's vibe, and CTAs for WhatsApp ordering and location check.
- Philosophy Section with Open-Air Vibe: Display tagline 'Handcrafted, Unexpected, Yours.' along with a descriptive copy about the coffee shop's authentic experience using an editorial layout with ample whitespace.
- Product Deep-Dive: Showcase 'BUTTERSCOTCH GLITCH' and 'STATIC BLACK' with high-contrast imagery and detailed taste profiles in grid boxes.
- Online Delivery Integration: Sections linking to GoFood and ShopeeFood with monochromatic icons for online orders.
- Instagram Gallery Grid: Display a 3x3 or 4x2 grid of recent Instagram photos related to the coffee shop.
- Grayscale Google Maps Embed: Embed a grayscale Google Maps with the location of the coffee shop, alongside an accordion-style FAQ for operating hours and order methods.

## Style Guidelines:

- Background: Deep Pitch Black (#000000) to set a minimalist, industrial tone.
- Text: Pure White (#FFFFFF) for high contrast and readability against the dark background.
- Grid Lines/Borders: Very Thin Light Grey (#333333) for subtle separation of sections.
- Headlines: 'Inter' (sans-serif) in bold and all caps for a modern, clean look. Note: currently only Google Fonts are supported.
- Details/Prices/Labels: 'Source Code Pro' (monospace) for small details, prices, and labels to maintain a consistent industrial aesthetic. Note: currently only Google Fonts are supported.
- Grid-based design with thin (1px) borders to separate sections horizontally and vertically, inspired by editorial layouts.
- Subtle 'Grain' overlay on the entire site to add texture.
- Add a 0.5s 'Glitch' transition effect on hover for images and buttons to enhance interactivity with Framer Motion.
